## Installation:

Zip-Verzeichnis in beliebigen Ordner entpacken. Eine Installation ist nicht notwendig.



## Update einer bestehenden Version:

Soll eine frühere Version des Programms aktualisiert werden,
genügt es die alte "Abrechnungsprogramm.exe"-Datei (Windows) bzw. "Abrechnungsprogramm.jar" (Linux / Mac) mit der Datei aus dem vorliegenden Zip-Archiv zu ersetzen.
Gegebenenfalls über den "Schicht-Update"-Button im Reiter "Info / Update" ein Update der Schichtliste ausführen.
